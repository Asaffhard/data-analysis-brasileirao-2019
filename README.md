# 📊 Análise do Campeonato Brasileiro 2019

## 🎯 Objetivo

Analisar dados do Brasileirão 2019 para identificar padrões e relações entre variáveis que influenciam o desempenho dos times.

## ⚽ Descrição

Este projeto realiza uma análise de dados do Campeonato Brasileiro de 2019, explorando relações entre desempenho dos times e variáveis como valor de mercado, reforços, idade média e número de gols.

## 🛠️ Tecnologias utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn

## 📊 Análises realizadas

* Correlação entre valor de mercado e pontos
* Impacto de reforços no desempenho
* Influência de jogadores estrangeiros
* Relação entre gols marcados e pontuação
* Análise de idade média dos times
* Estatísticas descritivas (média, mediana, desvio padrão)

## ▶️ Como executar

1. Instale as dependências:

```
pip install -r requirements.txt
```

2. Execute o script:

```
python src/cod_projeto_BR19.py
```
